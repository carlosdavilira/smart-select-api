# smart-select-api

To Install and run this project, you will need installed:
 - STS (Spring Tools Suit) installed.
 - MySQL DB - working on 3306 port.
 - Java version 1.8 
 - JDK version 1.8
 
 Other requirements:
Available ports : 3306, 8080 and 8081
